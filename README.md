# composition-el-view

使用Vue Composition Api配合Element UI动态配置创建响应式表单以及表格视图。

项目基于Vue 2.6.11以及Composition Api 0.5.0版本。

- [ ] 动态创建表单（进行中）
- [ ] 表单视图组件
- [ ] 响应式配置表单工具函数
- [ ] 表格视图
- [ ] 单元测试（进行中）
- [ ] Demo
